# fftw.cr

Crystal wrapper to the [FFTW 3](http://www.fftw.org) library

## Installation

First install fftw:

```bash
sudo pacman -S fftw
```

Add this to your application's `shard.yml`:

```yaml
dependencies:
  fftw.cr:
    github: firejox/fftw.cr
```

